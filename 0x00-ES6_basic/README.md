used. const and let are accepted but const is the more preferred of the
two

Do not allow change of variable within if block

### [2-arrow.js](./2-arrow.js)
Use arrow function

### [3-default-parameter.js](./3-default-parameter.js)
Use default parameter

### [4-rest-parameter.js](./4-rest-parameter.js)
Use rest parameter to make a variadic function

### [5-spread-operator.js](./5-spread-operator.js)
Use spread operator to concatenate arrays and strings
